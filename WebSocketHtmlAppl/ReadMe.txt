Softwares required:
1.GlishFish4.1.1 Server
2.NetBeans IDE

Steps to run web socket application
1.check out the code and save in local folder (ex: D:/WebSocket/)
2.open the NetBeansIDE 
3.open new project->select java web -> select web application with existing source -> click on next
4.Location : select the folder where you checked out the code (Ex:D:/WebSocket/WebSocketHtmlAppl)
  Project Name : WebSocketHtmlAppl
  Project folder : D:/WebSocketHtml2 (ex: select any local folder) 
  click on Next
5 select GlassFish server 4.1.1 click on next
6.WEB-INF content : browse web-inf content folder of D:/WebSocket/WebSocketHtmlAppl/web/WEB-INF
	click on finish
7. select the application-> right click ->click on run 
8.The application opens in browser
